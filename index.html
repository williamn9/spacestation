Design Framework

What is the Design System

The SpaceStation UX Design System is essentially a centralized, standardized, and reusable collection of user interface components (buttons, inputs, menbacus, etc.) visual styles, usage guidelines, and code that is all aligned. The system should be followed when building a business applications, so that our users can have a consistent experience when using our products.

From technical perspectives, using the Design System will speed up the development process by reusing components. Communications between design and development teams will be steamlined by using the same terminologies.

Material Design is selected among various publicly available Design Systems for its popularity, it is used by many Google products, and hence the users of those application would found the design familiar, making the app easier to learn. Also, having the support from a big corporation means better for continuity.


Design Principles
"Efficiency"
"Risk Reduction"

Design principles help to guide and steer a design throughout the design process. At each point of the process it should be possible to consider and review a design in the context of a shared set of principles.

For example, when we need to choose between a design that favors Efficiency and a design that favors Viewing Comfort, we will take the one for Efficiency.

Efficiency could means see more with less clicks and scroll.


Design and Component Guidance
